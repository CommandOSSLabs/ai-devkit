# SDL Phase 1 Scope

## Goal

Produce a feature specification that becomes the implementation source of truth.

## Primary Output

- Default path: `docs/specs/{NNNN}-{feature-name}/spec.md`

## Scope Guidance

- Include product context when it improves implementation clarity.
- Prioritize technical specification quality, boundaries, and decision traceability.
- Keep unresolved decisions explicit in `Open Points`.
